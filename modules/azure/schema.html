    Field

    Description

    firstseen

    Timestamp of when a sync job discovered this node

    lastupdated

    Timestamp of the last time the node was updated

    id

    The Azure Tenant ID number

(AzureTenant)-[RESOURCE]->(AzurePrincipal)
